# WhyMe 项目结构

## 项目概览

这是一个Android单词学习应用，使用Kotlin语言开发，采用MVVM架构模式。

## 目录结构

### 根目录
- `app/` - 应用主模块
- `docs/` - 项目文档
- `gradle/` - Gradle配置文件
- `.gitignore` - Git忽略文件
- `README.md` - 项目说明
- `build.gradle.kts` - 项目级构建脚本
- `gradle.properties` - Gradle属性配置
- `gradlew` - Gradle包装器脚本
- `gradlew.bat` - Windows版Gradle包装器脚本
- `settings.gradle.kts` - Gradle设置脚本
- `项目结构.md` - 项目结构文档

### 应用模块 (app/)

#### 数据层 (src/main/java/com/zhoulesin/whyme/data/)
- `local/` - 本地数据存储
  - `AppDatabase.kt` - 应用数据库定义

## 功能模块说明

1. **单词学习** - 核心功能，包括单词展示、学习、测验等
2. **词库管理** - 提供不同级别的词库选择
3. **学习统计** - 记录学习进度和效果
4. **收藏功能** - 允许用户收藏重要单词
5. **个人资料** - 展示用户学习数据和成就

## 技术栈

- Kotlin
- Jetpack Compose (UI)
- Room (数据库)
- ViewModel (MVVM架构)
- Dependency Injection (依赖注入)
- DataStore (偏好设置)

## 后续修改建议

1. **添加新功能** - 在对应功能模块下添加新的Screen和ViewModel
2. **修改数据模型** - 修改 `domain/model/` 下的模型文件
3. **修改业务逻辑** - 修改 `domain/usecase/` 下的用例文件
4. **修改数据存储** - 修改 `data/local/` 和 `data/repository/` 下的文件
5. **修改UI组件** - 修改 `ui/components/` 下的组件文件
6. **修改导航** - 修改 `ui/navigation/` 下的导航文件
