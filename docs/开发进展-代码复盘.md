# WhyMe 代码复盘（2026-04-14）

## 目前已完成的功能

- 登录与用户态：已接入本地登录状态持久化（DataStore），支持登录/退出与按用户切换本地数据库。
- 词库初始化：首次启动会从 `assets/words` 导入高中/四级/六级词库到 Room，并保留 `level` 字段（当前主流程暂未使用分级切换）。
- 主流程导航：已实现「今日 / 六级库 / 我的」底部主导航，学习流程可进入新词、复习、测验和单词详情。
- 新词学习：支持单词卡翻转、掌握度打分（Again/Hard/Good/Easy）、收藏、TTS 发音、学习完成态。
- 复习流程：支持待复习词拉取、掌握度更新、收藏、TTS 发音、复习完成态。
- 测验流程：支持选择题测验、答题反馈、结果页展示，并写入测试记录。
- 六级库页面：支持六级词表浏览、词库内搜索过滤、进度展示、跳转复习/测验。
- 收藏与详情：支持收藏列表搜索、取消收藏、详情页查看与收藏切换、详情页 TTS。
- 设置页：支持配置每日学习/复习/测试计划词数并持久化保存。
- 统计与个人中心：已接入基础统计卡片、学习概览、掌握度分布、成就展示框架。

## 还缺少/未完全打通的部分

- 周趋势数据是占位：统计页「本周趋势」目前是按“今天/昨天/几天前”生成空壳数据，未读取真实每日记录。
- 词库分级未对用户开放：代码层支持多级词库，但当前主流程固定在六级，切级入口未落地。
- 学习统计精度有待修正：`todayAccuracy` 计算逻辑偏粗糙（使用了已掌握数与今日复习数组合），不够准确。
- 复习记录链路未完整落库：有 `recordWordReview` 能力，但复习流程中未看到对应调用。

## 下一步建议（按优先级）

1. 完成设置页最小可用版本：支持修改每日新词/复习目标并写回 `PreferencesDataStore`。
2. 修正统计数据来源：将「本周趋势」改为读取 `DailyLearningRecord` 的真实数据。
3. 补齐复习记录与准确率口径：复习流程写入 review record，并统一“今日正确率”计算口径。
4. 评估并清理未接线页面/逻辑（如未进入主流程的学习中心、词库分级配置流），减少维护成本。
