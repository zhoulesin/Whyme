## 2. 关键数据流

### 2.1 首页数据流 (HomeScreen → HomeViewModel)

```
HomeScreen composable
    ↓
HomeViewModel.uiState (StateFlow<HomeUiState>)
    ↓
┌─────────────────────────────────────────────────────────────┐
│  combine(                                                   │
│    getUserStatsUseCase()      → userStats                  │
│    getDailyGoalUseCase()      → dailyGoal                  │
│    getTodayRecordUseCase()    → todayRecord                │
│    getWordsForReviewUseCase() → wordsForReview             │
│  )                                                         │
└─────────────────────────────────────────────────────────────┘
    ↓
显示: 新学单词数 / 复习完成数 / 连续打卡 / 正确率
```

### 2.2 学习会话数据流 (LearningSessionScreen → LearningViewModel)

```
LearningSessionScreen composable
    ↓ LaunchedEffect(Unit)
viewModel.initSession()
    ↓
viewModel.startLearning() / startReview() / startQuiz()
    ↓
LearningViewModel._uiState (StateFlow<LearningUiState>)
    ↓
_uiState.learningState: LearningState
    ├─ Idle
    ├─ Learning → 显示单词卡片 + 记忆按钮
    ├─ Testing → 显示测试题目
    ├─ Completed → 显示完成统计
    ├─ QuizResult → 显示测试结果
    └─ Error
    ↓
markWord(result) / nextQuestion()
    ↓
updateWordReviewUseCase(wordId, result)
    ↓
WordRepositoryImpl.updateWordReview()
    ↓
WordDao.updateWordReview()
    ↓
recordLearningSessionUseCase(...) ← [重要] 学习会话记录
    ↓
LearningRepositoryImpl.recordLearningSession()
    ↓
LearningRecordDao.insertRecord()
```

### 2.3 学习会话记录关键代码位置

**LearningViewModel.kt** (第 190-251 行 `markWord` 方法)：
- 调用 `updateWordReviewUseCase` 更新单词复习状态
- 学习完成时调用 `recordLearningSessionUseCase` 记录会话

**LearningRepositoryImpl.kt** (第 77-128 行 `recordLearningSession` 方法)：
- 获取或创建今日学习记录
- 更新 wordsLearned, wordsReviewed, correctCount, durationSeconds
- 调用 `saveRecord` 持久化
- 调用 `updateStreakIfNeeded` 更新连续打卡

**LearningRecordDao.kt**：
- `insertRecord` 使用 `@Insert(onConflict = OnConflictStrategy.REPLACE)`
- 查询使用 `LocalDate.now().toEpochDay()` 作为日期标识

## 3. 关键状态定义

### 3.1 LearningState (domain/model/LearningState.kt)

```kotlin
sealed class LearningState {
    data object Idle : LearningState()
    data class Learning(...) : LearningState()      // 学习/复习中
    data class Testing(...) : LearningState()       // 测试中
    data class Completed(...) : LearningState()      // 学习完成
    data class QuizResult(...) : LearningState()     // 测试完成
    data class Error(...) : LearningState()
}
```

### 3.2 UserStats (domain/model/LearningRecord.kt)

```kotlin
data class UserStats(
    val totalWordsLearned: Int,       // 累计已学单词
    val totalWordsReviewed: Int,      // 累计复习单词
    val currentStreak: Int,            // 连续打卡天数
    val longestStreak: Int,
    val totalLearningMinutes: Long,     // 总学习时长
    val todayWordsLearned: Int,         // 今日已学
    val todayWordsReviewed: Int,       // 今日已复习
    val todayAccuracy: Float            // 今日正确率
)
```

## 5. 导航结构

### 5.1 路由定义 (ui/navigation/Screen.kt)

```kotlin
sealed class Screen(val route: String) {
    data object Home : Screen("home")              // 首页 (底部Tab)
    data object Learning : Screen("learning")      // 学习中心 (底部Tab)
    data object LearningSession : Screen("learning/session/{mode}")  // [二级页面] 学习会话
    data object Profile : Screen("profile")      // 我的 (底部Tab)
    data object Favorites : Screen("favorites")   // 收藏页面 (二级)
    data object WordDetail : Screen("word/{wordId}")
}

/**
 * 学习模式
 */
enum class LearningModeType {
    AUTO,       // 自动选择（复习优先，否则学习）
    LEARN,      // 学习新词
    REVIEW,     // 复习

}
```

### 5.2 导航流程

```
首页 [开始学习] ──▶ learning/session/AUTO
    │                        │
    │              自动选择复习或学习模式
    │
学习中心Tab ──▶ LearningScreen (入口)
    │
    ├─ 学习 ──▶ learning/session/LEARN
    ├─ 复习 ──▶ learning/session/REVIEW
    └─ 测试 ──▶ learning/session/QUIZ
```

## 6. 数据持久化关键点

### 6.1 学习会话记录时机

**重要**: 数据必须在导航离开前保存完成！

```kotlin
// LearningViewModel.kt - markWord() 方法
if (nextIndex >= currentState.total) {
    // 学习完成 - 必须先保存数据
    kotlinx.coroutines.runBlocking {
        recordLearningSessionUseCase(
            wordsLearned = finalStats.wordsLearned,
            wordsReviewed = finalStats.wordsReviewed,
            correctCount = finalStats.correctCount,
            durationSeconds = durationSeconds
        )
    }
    // 然后更新状态显示完成界面
    _uiState.update {
        it.copy(learningState = LearningState.Completed(...))
    }
}
```

### 6.2 DataStore 持久化标记

**PreferencesDataStore.kt**：
- `word_database_initialized` - 词库是否已初始化
- `current_streak` / `longest_streak` - 连续打卡记录
- `last_learning_date` - 上次学习日期

## 7. 常见问题排查

### 7.1 首页数据为 0
- 检查 `LearningRepositoryImpl.recordLearningSession` 是否被调用
- 检查 `LearningRecordDao.insertRecord` 是否使用 `REPLACE` 策略
- 检查 Flow 是否正确 emit 新数据

### 7.2 学习会话数据不更新
- 确认 `runBlocking` 或 `await` 等待数据保存完成
- 检查 `exitSession()` 是否在导航前调用

### 7.3 单词收藏状态不更新
- 检查 `toggleFavoriteUseCase` 实现
- 检查 `LearningViewModel.toggleFavorite` 中的手动状态更新
