## 1. 概述

本文档定义 WhyMe English 应用的学习流程设计，涵盖词汇管理、学习模式、记忆曲线复习、测试评估等核心功能。

### 1.1 核心设计原则
1. **科学记忆**：基于艾宾浩斯遗忘曲线安排复习
2. **循序渐进**：新词学习 → 短期记忆 → 长期记忆
3. **即时反馈**：每次学习都有明确的效果反馈
4. **灵活可控**：支持中断、暂停、切换模式

## 2. 词汇库管理

### 2.1 内置词库

| 词库名称 | 文件 | 词数 | 难度等级 |
|---------|------|------|---------|
| 高中词汇 | `GaoZhong.json` | ~5000 | 基础 |

| GRE | `GRE.json` | ~4000 | 精通 |

### 2.2 单词数据模型

```kotlin
@Entity(tableName = "words")
data class WordEntity(
    @PrimaryKey(autoGenerate = true)
    val id: Long = 0,

    // 基本信息
    val word: String,                    // 单词本体
    val phonetic: String,                // 音标（美式）
    val usPhonetic: String?,             // 美式音标
    val ukPhonetic: String?,             // 英式音标
    val definition: String,              // 英文释义
    val translation: String,             // 中文翻译

    // 例句与短语
    val phrases: String?,                // JSON: [{phrase, translation}]
    val sentences: String?,              // JSON: [{sentence, translation, audio?}]

    // 学习状态
    val masteryLevel: Int = 0,           // 掌握等级 0-5
    val isFavorite: Boolean = false,     // 是否收藏
    val isLearned: Boolean = false,      // 是否已学习过
    val isNew: Boolean = true,           // 是否新词

    // 记忆曲线
    val nextReviewDate: Long? = null,    // 下次复习日期 (Epoch day)
    val reviewCount: Int = 0,            // 复习次数
    val correctCount: Int = 0,           // 正确次数
    val lastReviewResult: ReviewResult? = null, // 上次复习结果

    // 时间戳
    val createdAt: Long = System.currentTimeMillis(),
    val learnedAt: Long? = null,          // 首次学习时间
    val lastReviewedAt: Long? = null,   // 上次复习时间

    // 关联词库
    val wordBank: String? = null         // 来源词库
)
```

## 3. 学习模式

### 3.1 三大学习模式

| 模式 | 说明 | 触发条件 |
|-----|------|---------|
| **新词学习** | 学习尚未掌握的单词 | 用户主动触发 / 每日任务 |
| **复习巩固** | 复习已学但需要巩固的单词 | 达到复习日期 / 用户主动触发 |
| **测试挑战** | 检验学习成果 | 用户主动触发 |

## 4. 记忆曲线算法

### 4.1 艾宾浩斯遗忘曲线

基于艾宾浩斯遗忘曲线，单词复习遵循以下间隔：

| 复习次数 | 遗忘曲线间隔 | 推荐复习时机 |
|---------|-------------|-------------|
| 1 | 1天 | 首次学习后 |
| 2 | 2天 | 第2天 |
| 3 | 4天 | 第4天 |
| 4 | 7天 | 第7天 |
| 5 | 15天 | 第15天 |
| 6 | 30天 | 第30天 |
| 7 | 永久掌握 | 移出待复习队列 |

### 4.2 间隔调整规则

```
基础间隔 = 根据 masteryLevel 计算

根据 ReviewResult 调整：
- AGAIN: 间隔 × 0.5，下一个立即出现
- HARD:  间隔 × 1.0，保持原计划
- GOOD:  间隔 × 1.2，适当延长
- EASY:  间隔 × 2.0，大幅延长

每日学习上限：
- 新词学习：最多 20 个/天
- 复习单词：最多 50 个/天
```

### 4.3 掌握等级

```kotlin
enum class MasteryLevel(val description: String, val intervalDays: Int) {
    UNKNOWN(0, "陌生", 0),        // 未学习
    LEARNING(1, "学习中", 1),    // 刚学习
    FAMILIAR(2, "熟悉", 2),       // 短期记忆
    UNDERSTOOD(3, "理解", 4),     // 中期记忆
    MASTERED(4, "掌握", 7),       // 长期记忆
    PERFECTED(5, "精通", 15)      // 深度记忆
}
```

## 6. 数据统计

### 6.1 统计维度

| 维度 | 指标 | 计算方式 |
|-----|------|---------|
| **今日** | 学习单词数 | 当日学习 + 复习 |
| | 正确率 | 正确数 / 总答题数 |
| | 学习时长 | 会话时长累加 |
| **本周** | 总学习单词 | 周累计 |
| | 连续学习天数 | 最长连续 |
| | 平均正确率 | 周平均 |
| **累计** | 掌握单词数 | masteryLevel >= 3 |
| | 总复习次数 | 全部复习记录 |
| | 精通单词数 | masteryLevel = 5 |

## 8. 技术实现要点

### 8.1 关键组件

| 组件 | 职责 |
|-----|------|
| `AppInitializer` | 应用启动时初始化词库 |

### 8.2 协程使用

```kotlin
// 学习流程使用 viewModelScope
viewModelScope.launch {
    // 1. 加载待学习单词
    val wordsToLearn = getWordsForLearningUseCase()

    // 2. 逐词学习
    wordsToLearn.forEach { word ->
        // 显示卡片，等待用户操作
        val result = waitForUserInput()

        // 3. 记录结果（IO操作）
        updateWordReviewUseCase(word.id, result)

        // 4. 更新记忆曲线
        calculateNextReviewDate(word, result)
    }

    // 5. 保存会话统计
    saveSessionResults()
}
```

### 8.3 状态持久化

```kotlin
// 退出时保存进度
override fun onPause() {
    super.onPause()
    if (learningState is LearningState.Learning) {
        // 保存当前进度到 DataStore
        saveLearningProgress(currentIndex, remainingWords)
    }
}

// 恢复时读取进度
init {
    val savedProgress = loadLearningProgress()
    if (savedProgress != null) {
        // 恢复之前的进度
        restoreProgress(savedProgress)
    }
}
```
